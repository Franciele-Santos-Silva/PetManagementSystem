Este projeto é um CRUD de cadastro de pets implementado em Java.

## Tecnologias utilizadas
- Java 8+  
- JDBC  
- Banco de dados relacional MySQL  
- IDE: IntelliJ IDEA  
- Maven (opcional) para gerenciamento de dependências

## Métodos implementados
- **inserir(Pet pet):** adiciona um novo pet no banco de dados  
- **listar():** retorna uma lista com todos os pets cadastrados  
- **deletar(int id):** remove um pet do banco de dados pelo ID  
- **atualizar(int id, Pet pet):** atualiza os dados de um pet existente pelo ID
